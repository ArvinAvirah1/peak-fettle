The set-logging screen is now the Focus Stepper. Colour is held constant on purpose — these frames show the interaction structure you specified: the dynamic "Continue to <next>" + "Select different exercise" buttons, the exercise switcher (all routine exercises, done ones marked, plus Browse), the "ask where to place it" prompt for off-routine adds, two ways to manage routines for you to compare, and three ways the stepper can work for non-routine users for you to pick. Maps to TICKET-059…062.
