1. **Source of truth** — Actual codebase files to read
2. **Learning outcomes** — Bloom-tagged (L1–L5)
3. **Pre-lesson survey (M1)** — 3 diagnostic questions
4. **Spacing carry-over (M14)** — Connections to prior lessons
5. **Difficulty ladder (M2)** — 5–6 graduated rungs
6. **Concept sequence** — 6–9 core concepts, each with:
   - (M4) Generate-first question
   - (M7) Concrete hook with examples
   - **Real code** from the actual codebase
   - (M6) Elaboration & nuance
   - (M3) Retrieval check
   - (M9/M12) Checkpoint exercises
7. **Teach-back (M10)** — 5-sentence synthesis
8. **Cumulative review (M13)** — Rapid-fire questions
9. **Graded quiz** — Bloom L1–L5 with:
   - Question prompt
   - Rubric & point value
   - Model answer
   - Full assessment guidance
10. **Interactive widget** — Specialized visualization per lesson
11. **End-of-session updates** — Agent instructions for grading & LEARNER_PROFILE
