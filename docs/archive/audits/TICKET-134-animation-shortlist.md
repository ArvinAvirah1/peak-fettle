# TICKET-134 -- v2 animation shortlist (exercise media)

**Status: cue content pending founder review.** Every cue in `exerciseCatalog.ts`
(both the original 51 engine-catalog entries and the 181 TICKET-134 media-only
additions) is bulk-authored, coaching-claim content. Per the ticket's
acceptance criterion #4, **none of it ships to real users until the founder
has reviewed it** -- these are read as instructional advice, not just labels,
and a wrong cue is a liability, not a typo. This document itself is scoped
only to identifying WHICH exercises need more than text in v2; it does not
change that review requirement.

## Purpose

v1 ships static muscle diagrams + 3 written cues per exercise (no video, no
new art pipeline). For most exercises that's enough -- the bar path is a
straight vertical line, or the movement is simple enough that "brace, lower,
drive" reads clearly. For a smaller set, the actual skill is a *path* or
*timing* problem that words describe poorly (an Olympic pull, a rotational
bar path, a machine most users have never seen). This is the shortlist for
that smaller set -- the natural v2 backlog when licensed WebP/Lottie loops
are procured (a separate, later content-procurement decision per the ticket's
notes; licensing is explicitly out of scope for this ticket).

## Shortlist (~20 exercises)

1. **Snatch** -- full pull + turnover + overhead catch is a multi-phase bar
   path that cues alone chronically under-describe; highest injury/technical
   stakes in the catalog.
2. **Power Snatch** -- same turnover complexity as Snatch, shorter pull; still
   a 3D bar path (close to body, then punch under).
3. **Hang Snatch** -- same catch mechanics as Snatch from a different start
   position; the "hang" starting height itself is hard to convey in text.
4. **Clean** -- the pull-to-rack transition (elbows whipping around to the
   front rack) is the single hardest-to-verbalize part of any lift here.
5. **Power Clean** -- same rack-transition problem as Clean, abbreviated pull.
6. **Hang Clean** -- rack transition again, plus a hang-height starting cue
   that's genuinely a spatial/visual concept.
7. **Clean and Jerk** -- two multi-phase lifts back to back (clean's rack
   transition + jerk's dip-drive-split/squat-under); the compound movement
   with the most sequential sub-skills in the catalog.
8. **Jerk** -- the dip-drive-punch-under-split (or squat) timing is a
   fractions-of-a-second coordination skill; text cues can describe the
   phases but not the timing feel.
9. **Overhead Squat** -- the squat itself is simple, but keeping the bar
   stacked over the ears while descending is a balance/proprioception skill
   best shown, not told.
10. **Turkish Get-Up** -- a 7-step sequenced movement through multiple
    positions (lying, elbow, hand, bridge, kneel, stand); "move through each
    step slowly" is honest but not sufficient instruction on its own.
11. **Dragon Flag** -- an advanced core movement where the failure mode
    (hips sagging, using momentum) is much easier to self-correct watching a
    demo than reading about it.
12. **Nordic Curl** -- eccentric-only control at a joint angle most users have
    never trained; the "how slow is slow enough" pacing is a visual/tempo
    concept.
13. **Glute-Ham Raise** -- same eccentric-control problem as Nordic Curl, on
    a piece of equipment (GHR bench) many users have literally never seen.
14. **Pistol Squat** -- a balance-dependent unilateral squat where the failure
    mode (falling backward, knee cave) benefits enormously from watching
    correct hip/ankle mobility being used.
15. **Cuban Press** -- an uncommon 3-phase shoulder movement (external
    rotation -> press -> reverse) that most users have never seen named, let
    alone performed; "rotate so forearms point overhead" is a hard mental
    image to build from text alone.
16. **Landmine Row (and Meadows Row)** -- the landmine's fixed-arc bar path
    around a pivot point is a novel piece of equipment for most home-gym /
    new-to-a-format users; seeing the arc once removes most confusion.
    *(Grouping these two together -- same equipment novelty, same fix.)*
17. **Sissy Squat** -- a balance + ankle-lock movement most users have never
    attempted; the straight-line torso descent is very easy to get visually
    wrong (turning it into a bad leg extension) without a reference video.
18. **Reverse Hyperextension** -- equipment novelty (few commercial gyms have
    a dedicated reverse hyper) plus a leg-swing timing/ROM that's easy to
    misjudge from text.
19. **Tire Flip** -- the squat-lift-then-flip-through transition (hand
    re-positioning under the tire mid-rep) is a full-body coordination
    sequence that reads as multiple disconnected steps in text.
20. **Battle Ropes** -- the intended "wave" pattern (alternating vs. double,
    amplitude, rhythm) varies by coaching style and is fundamentally a
    rhythm/timing skill that a short text cue cannot standardize.

## Notes for v2 scoping

- These 20 are a rationale-driven shortlist, not a hard cutoff -- if founder
  review of the v1 cues surfaces additional exercises where testers report
  confusion, add them here before licensing v2 content.
- Priority order roughly follows injury/technical stakes first (Olympic
  lifts), then equipment/skill novelty (GHR, landmine, reverse hyper,
  sissy squat), then rhythm/timing movements (battle ropes) -- useful if v2
  can only fund a subset of loops initially.
- No licensing or content-procurement work has started; this list only
  identifies WHERE video/animation would outperform text, per the ticket's
  scope boundary ("do not start licensing inside this ticket").
