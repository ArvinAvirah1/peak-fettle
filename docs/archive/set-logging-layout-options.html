Peak Fettle — set-logging layout options (structure only)

Six different ways to organise the same screen. Colour is held constant on purpose — the variable here is structure: where routines/splits live, how you reach a specific exercise, where the PB/rep-range context sits, and where the primary "Log set" action lands. Every layout keeps the three non-negotiables: routines & PPL/Upper-Lower splits one tap away, PB-on-select context, and a full-width primary action. Pick a structure (or combine) for TICKET-057.
